# Project Notes

## Overview
This project is a quadruped robot based on STM32 and PCA9685, designed to control 12 servos and perform basic walking movements.

## Main Work
- STM32 control development
- PCA9685 integration
- servo control testing
- gait adjustment
- movement testing
- practical debugging

## Practical Challenges
One of the main challenges was achieving stable movement with multiple servos working together.  
Mechanical constraints, servo direction, calibration differences, and body balance all affected the final walking performance.

Another important difficulty was adjusting gait behavior so that the robot could move forward in a more stable and practical way.

## What I Learned
This project helped me better understand:
- embedded control with STM32
- multi-servo coordination
- hardware and software integration
- practical debugging in robotics
- the effect of mechanical structure on real movement

## Possible Improvements
- better gait optimization
- improved stability
- cleaner mechanical structure
- more advanced motion control
